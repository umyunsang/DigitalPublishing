# Figma Static Workflow Notes

## With Figma MCP

- Use `get_design_context` for selected Figma frames when available.
- Prefer selected frames or components over entire heavy files.
- Extract layout, styles, component names, assets, and token hints.
- Convert the context into a static layout spec before writing HTML/CSS.

## Without Figma MCP

- Ask for a screenshot, exported image, or copied frame details.
- State which assumptions are inferred from the image.
- Keep the output as a handoff spec when implementation is not requested.

## Course Constraint

The goal is design handoff and static layout practice. Do not turn the task into a production app unless the user explicitly asks.
