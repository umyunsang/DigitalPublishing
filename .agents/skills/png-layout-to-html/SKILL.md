# Png Layout To Html

## Overview

Convert an annotated layout image into a static Digital Publishing exercise. Treat the image annotations as requirements and preserve the course's fixed-width/static design intent.

## Workflow

1. Inspect the image before editing files.
2. Extract explicit measurements, zones, text hierarchy, asset notes, link behavior, and visual effects.
3. Identify uncertain details such as missing image filenames or exact text.
4. Create a short layout interpretation before coding when assumptions matter.
5. Implement standalone HTML/CSS with semantic structure.
6. Preserve annotated fixed widths, menu alignment, hover rules, card radius, shadows, and footer styling.
7. Avoid frameworks and build tools.

## Output

When planning:

- Extracted layout spec
- File plan
- CSS technique choices
- Missing assets/questions

When implementing:

- HTML file
- CSS file or inline CSS, matching the assignment request
- Brief verification notes
