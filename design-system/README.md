# Design System Starter

This folder is a small token reference for static Digital Publishing exercises.

Use `tokens.json` to keep visual decisions explicit before writing CSS. The format follows the Design Tokens Community Group style with `$type` and `$value` fields.

## Token Categories

- `color`: background, text, accent, surface, border.
- `space`: spacing scale for layout rhythm.
- `font`: Korean-first font stack.
- `size`: type scale.
- `radius`: border radius choices.
- `shadow`: subtle depth options.

## How to Use

1. Pick a small set of tokens before coding.
2. Translate them into CSS custom properties or normal CSS values.
3. Do not invent a new palette for every page unless the assignment requires it.
4. Update this file when a course-wide visual rule becomes stable.
