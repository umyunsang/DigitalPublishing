KOSMOS Agent Stack / 핵심 에이전트 소개

  • Query Engine

    민원 상태머신을 관리하고 부처별 경로로 요청을 라우팅하는 핵심 엔진입니다. 대화 맥락을 구조화된 액션 트리로 변환합니다.

  • Tool Factory

    공공 OpenAPI 명세를 읽어 타입을 갖춘 툴을 자동 생성하는 어댑터 팩토리입니다. 5,000여 공공 API를 일관된 인터페이스로 노출합니다.

  • Permission

    인증과 개인정보 마스킹, 감사 로그를 단계별로 누락없이 처리합니다. 모든 접근은 원칙 기반 파이프라인을 통과합니다.

  • Orchestrator

    K-EXAONE 236B 기반으로 전체 계획을 수립하고 다른 에이전트를 지휘합니다. 복잡한 민원을 하위 태스크로 분해·병합합니다.

KOSMOS는 Korean public Service Multi-agent Orchestration System의 약자로, data.go.kr에 공개된 5,000여 개의 공공 OpenAPI를 단일 대화형 인터페이스로 통합하는 학술 R&D 프로젝트입니다. 국민은 복잡한 부처 경계를 인지하지 않고, 자연어 한 줄로 교통 안전 정보 조회, 복지 수급 신청, 주소 이전, 긴급 서비스 호출 같은 크로스-부처 민원을 해결할 수 있습니다.

시스템은 Query Engine, Tool Factory, Permission Pipeline, Agent Swarm, Context Memory, Error Recovery 여섯 계층을 공공 도메인으로 재설계했으며, 추론 엔진으로는 LG AI Research의 K-EXAONE 모델군을 채택합니다. Apache License 2.0 하에 공개되며, 어떠한 상용 기관과도 관계가 없는 독립 학술 프로젝트입니다.
