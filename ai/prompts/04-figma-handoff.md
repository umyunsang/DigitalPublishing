# Prompt: Figma to Static Handoff

Use this when a Figma frame, screenshot, or Figma MCP design context is available.

Task:
1. Identify the selected frame or screenshot scope.
2. Extract content hierarchy, layout, spacing, typography, color, and assets.
3. Convert design context into a static layout spec.
4. Note which design decisions should become CSS and which should remain Figma-only annotations.
5. Preserve the course requirement: static HTML/CSS first, no framework unless requested.

Output:
- Design context summary
- Layout spec
- Asset list
- HTML/CSS implementation notes
- Review checklist
