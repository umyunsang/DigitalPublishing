# Prompt: PNG Layout Blueprint to HTML/CSS

You are helping with a Digital Publishing class exercise. The user provides a PNG layout blueprint with red annotations, measurements, style notes, and content requirements.

Task:
1. Read the PNG as a layout specification, not just a visual reference.
2. Extract measurements, layout zones, text hierarchy, link behavior, image requirements, hover states, radius, shadow, and footer details.
3. List assumptions if the PNG is ambiguous.
4. Produce a static HTML/CSS implementation plan.
5. Create standalone HTML/CSS only when requested.
6. Do not introduce React, Tailwind, build tools, or backend code.

Output:
- Extracted layout spec
- File plan
- HTML/CSS implementation notes
- Missing assets or questions

When coding, preserve annotated fixed widths such as `1400px`, right-aligned menu rules, menu hover color differences, card radius/shadow, and vertical scroll notes.
