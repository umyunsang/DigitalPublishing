# Prompt: Design Token Audit

Audit the visual decisions in a static page or Figma design.

Task:
1. Extract observed colors, spacing, type sizes, radius, and shadow values.
2. Compare them with `design-system/tokens.json`.
3. Identify arbitrary or duplicated values.
4. Recommend a minimal token set for the page.
5. Do not over-systematize one-off class exercises.

Output as a table with observed value, proposed token, and reason.
